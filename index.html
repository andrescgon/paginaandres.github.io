Productos

Imágenes: Para que la IA funcione de una manera óptima se le tiene que dar información principalmente un conjunto de imágenes las cuales la IA va reconociendo y siguiendo diferentes parámetros para facilitar su entrenamiento como: el color, tamaño, forma, figura...etc.

Posturas: Para el apartado de posturas la IA lee el cuerpo y dependiendo de cómo lo vea le crea un esqueleto para entender que movimientos se están realizando, por medio de esas posturas las cuales va guardando hacen que la IA reconozca fácilmente la postura que se quiere identificar.

Sonidos: Para el apartado de sonidos ya la situación es más técnica puesto a que la IA aprovecha tanto altos como bajos y timbre para poder reconocer el sonido el cual se está tratando identificar aparte de seguir un patrón entre las muestras de data que se le dan puede identificar de manera eficiente los sonidos que se quieran identificar.
